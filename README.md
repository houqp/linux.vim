# linux.vim

Vim plugin that prepare your vim for Linux Kernel hacking.


#Installation
Use `Pathogen`, `Vundle` or just put the files into your `~/.vim` directory.


#Usage
Use `:ToggleLinuxKernelCoding` command to toggle the kernel hacking settings.
